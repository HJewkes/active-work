# Dogfood: active-work's own initiative

This directory is a copy of the active-work initiative directory
for THIS project, scaffolded with `aw new`. It demonstrates the
on-disk shape and serves as the v0.1 wishlist (8 tasks).

To make this your operating directory: `aw setup` (creates a fresh
active root in `$XDG_DATA_HOME/active-work/`).
